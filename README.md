# Vesper Painting Website

Static single-page website for Vesper Painting.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Run locally

Open `index.html` directly in a browser, or serve the directory:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

Deploy as a static site on any host (Netlify, Vercel static, GitHub Pages, S3/CloudFront, etc.).
